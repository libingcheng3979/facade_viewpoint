# Facade Viewpoint

## 项目简介
针对传统街景获取中“视角偏差大”与“数据覆盖低”的难题，本项目提出了一种基于计算几何与语义约束的自动化街景采样点生成方法。

## 功能特性
- **数据清洗**：自动修复无效几何，过滤小面积建筑。
- **语义筛选**：支持排除高速公路、人行道等不适合街景采集的道路类型。
- **几何计算**：基于建筑轮廓计算最佳拍摄角度 (Heading) 和距离。
- **可视化**：生成交互式地图、统计图表及采样验证图。

## 目录结构
- `src/`: 核心源代码
- `data/`: 输入输出数据 (需自行放入 GeoJSON 文件)
- `notebooks/`: 实验性代码

## 使用方法
1. 安装依赖: `pip install -r requirements.txt`
2. 修改 `src/config.py` 中的数据路径和相关配置。
3. 运行: `python main.py`
